# Cargo Rune `/run`

This directory exists as a basic testing ground for the command line utility to ensure that it works properly and actually runs things. It also serves as a really basic example repository for rune projects.

This directory is inherently volatile and there's no guarantee that it will not significantly change between commit versions, but nonetheless an effort should be made to try not to break things too much.

Merge conflicts within this directory can be resolved bluntly and with force if necessary.
